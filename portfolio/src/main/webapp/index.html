Projects

Pong Game

This was a final project for my Introduction to Software Engineering class and it consisted of my group and I building a physics engine and using that engine to build a game. The entire project was done in C and used the OpenGL graphics library. The game we chose to create was a version of the game pong where one can chose to play with any number of AIs in a single game (the players are organized around an appropriate regular polygon) and with one other person either on the same keyboard or over a network. My responsibilities for the project were to implement the actual physics for the engine and add the network functionality.

Fast Hamiltonian Simulation

This was a final project for my Quantum Programming and Algorithms course and it consisted of implementating an algorithm described in this paper called qDRIFT. Our task was to compare the time complexity of qDRIFT to more standard methods of compiling quantum circuits (i.e. First Order Trotter-Sizuki) with respect to metrics such as desired precision of simulation, simulation time, and length of hamiltonian decomposition used in the simulation. A final paper was written decribing the results obtained from this project. The paper and all the code (written in Python and Microsoft's quantum development language Q#) can be found in this repository.
